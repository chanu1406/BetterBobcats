## üìÖ Timeline

### Sophomore Year
- Complete CSE 150, 160
- Join security club
- Start HackTheBox/TryHackMe
- Learn Linux basics

### Junior Year
- Complete CSE 178, 130, 031, 120 (Tier 1)
- Get Security+ certification
- Participate in 2+ CTFs
- Apply for security internships
- Start building portfolio

### Senior Year
- Complete Tier 2 courses (3-4)
- Complete security internship
- Build substantial portfolio
- Consider CEH or OSCP prep
- Network with security professionals

## üéØ Quick Tips

1. **Start Early**: Security builds on fundamentals - don't wait until senior year
2. **Practice Ethically**: Only hack authorized systems (CTFs, bug bounties with permission)
3. **Build Portfolio**: Document CTF solutions, create security tools, write blog posts
4. **Join Community**: r/netsec, security Discord servers, local BSides conferences
5. **Stay Current**: Follow security news, CVEs, new vulnerabilities
6. **Get Certified**: At minimum Security+ before graduation
7. **Network**: Connect with security professionals, attend conferences
8. **Think Like Both**: Learn both offensive (pentesting) and defensive (SOC) perspectives
